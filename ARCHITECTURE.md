═══════════════════════════════════════════════════════════════
                        DATA FLOW
═══════════════════════════════════════════════════════════════

1. Пользователь открывает главную страницу (/)
   └─> Projects компонент делает GET запрос к API
       └─> API читает db.json
           └─> Возвращает список проектов
               └─> Отображаются на странице

2. Администратор хочет добавить проект
   └─> Переходит на /login
       └─> Вводит пароль (admin123)
           └─> Получает доступ к /admin
               └─> Заполняет форму
                   └─> POST запрос к API
                       └─> API добавляет в db.json
                           └─> Проект появляется на главной

3. Редактирование проекта
   └─> Нажимает "Редактировать" в админке
       └─> Заполняет форму с текущими данными
           └─> PUT запрос к API
               └─> API обновляет db.json
                   └─> Изменения видны на главной

4. Удаление проекта
   └─> Нажимает "Удалить" в админке
       └─> DELETE запрос к API
           └─> API удаляет из db.json
               └─> Проект исчезает с главной
